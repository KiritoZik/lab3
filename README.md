## Описание реализованных команд

### Алгоритмы вычислений

#### `factorial` - вычисление факториала числа

- **Использование:** `factorial <число> [--recursive | -r]`

- **Описание:** Вычисляет факториал заданного неотрицательного целого числа. Поддерживает два режима: итеративный (по умолчанию) и рекурсивный (с опцией `-r`).

- **Примеры:**
  - `python -m src.main factorial 5` - вычисление факториала 5 итеративным методом
  - `python -m src.main factorial 5 --recursive` - вычисление факториала 5 рекурсивным методом

#### `fibonacci` - вычисление числа Фибоначчи

- **Использование:** `fibonacci <номер> [--recursive | -r]`

- **Описание:** Вычисляет n-ое число Фибоначчи (нумерация начинается с 0). Поддерживает два режима: итеративный (по умолчанию) и рекурсивный с мемоизацией (с опцией `-r`).

- **Примеры:**
  - `python -m src.main fibonacci 10` - вычисление 10-го числа Фибоначчи итеративным методом
  - `python -m src.main fibonacci 10 --recursive` - вычисление 10-го числа Фибоначчи рекурсивным методом

### Алгоритмы сортировки (только для чисел)

#### `counting_sort` - сортировка подсчетом

- **Использование:** `counting-sort <список_чисел>`

- **Описание:** Сортирует список неотрицательных целых чисел методом подсчета.

- **Пример:** `python -m src.main counting-sort 4 2 7 1 9`

#### `radix_sort` - поразрядная сортировка

- **Использование:** `radix-sort <список_чисел>`

- **Описание:** Сортирует список неотрицательных целых чисел поразрядной сортировкой.
- **Пример:** `python -m src.main radix-sort 170 45 75 90 2 802 24 66`

#### `bucket_sort` - блочная сортировка

- **Использование:** `bucket-sort <список_чисел> [--buckets <количество> | -b <количество>]`

- **Описание:** Сортирует список чисел с плавающей точкой блочной сортировкой. Количество блоков можно задать вручную (по умолчанию равно количеству элементов).

- **Примеры:**
  - `python -m src.main bucket-sort 0.42 0.32 0.33 0.52 0.37 0.47 0.51`
  - `python -m src.main bucket-sort 0.42 0.32 0.33 --buckets 3`
  - `python -m src.main bucket-sort -- -0.5 -0.3 -0.1 0.2 0.4` - работает с отрицательными числами

Примечание: для ввода отрицательных чисел необходимо указать -- до отрицательных чисел

### Алгоритмы сортировки (для чисел и строк)

#### `bubble_sort` - сортировка пузырьком

- **Использование:** `bubble-sort <список>`

- **Описание:** Сортирует список целых чисел или строк методом пузырька.

- **Примеры:**
  - `python -m src.main bubble-sort 64 34 25 12 22 11 90`
  - `python -m src.main bubble-sort banana apple cherry`

#### `quick_sort` - быстрая сортировка

- **Использование:** `quick-sort <список>`

- **Описание:** Сортирует список целых чисел или строк быстрой сортировкой (quicksort).

- **Примеры:**
  - `python -m src.main quick-sort 64 34 25 12 22 11 90`
  - `python -m src.main quick-sort banana apple cherry`

#### `heap_sort` - пирамидальная сортировка

- **Использование:** `heap-sort <список>`

- **Описание:** Сортирует список целых чисел или строк пирамидальной сортировкой.

- **Примеры:**
  - `python -m src.main heap-sort 64 34 25 12 22 11 90`
  - `python -m src.main heap-sort banana apple cherry`

### Структуры данных

#### `stack` - интерактивная работа со стеком

- **Использование:** `stack`

- **Описание:** Запускает интерактивный режим работы со стеком. Поддерживает следующие команды:
  - `push` - добавить элемент в стек
  - `pop` - получить и удалить верхний элемент
  - `peek` - получить верхний элемент без удаления
  - `is_empty` - проверить стек на пустоту
  - `min` - получить минимальный элемент стека
  - `len` - получить размер стека
  - `info` - вывести справку по командам
  - `quit` - выйти из интерактивного режима

- **Примечание:** Стек поддерживает хранение элементов одного типа (числа или строки).

#### `queue` - Интерактивная работа с очередью

- **Использование:** `queue`

- **Описание:** Запускает интерактивный режим работы с очередью. Поддерживает следующие команды:
  - `enqueue` - добавить элемент в очередь
  - `dequeue` - получить и удалить нижний элемент из очереди
  - `front` - получить нижний элемент без удаления
  - `is_empty` - проверить очередь на пустоту
  - `info` - вывести справку по командам
  - `quit` - выйти из интерактивного режима


- **Примечание:** Очередь поддерживает хранение элементов одного типа (числа или строки).

## Обработка ошибок

Все команды обрабатывают следующие типы ошибок:

- `ValueError` - невалидные параметры (отрицательные числа для факториала/Фибоначчи, отрицательные числа для counting_sort/radix_sort)
- `IndexError` - попытка получить элемент из пустого стека/очереди
- `TypeError` - несовпадение типов элементов в структурах данных

## Особенности реализации

### Рекурсивные алгоритмы

- **Фибоначчи:** Рекурсивная реализация использует `@lru_cache` для оптимизации повторных вычислений.

### Алгоритмы сортировки
- Все строки, которые могут принимать строки, воспринимают два типа данных:
str, int - то есть float будет приниматься за str.
- Быстрая сортировка использует медианный элемент как опорный для лучшей производительности.
